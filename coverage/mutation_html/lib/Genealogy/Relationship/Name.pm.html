 2178: C<new()>.  On error, this module simply calls C<< $logger->error($msg) >>
 2179: and Log::Abstraction handles ctx forwarding, formatting, and dispatch.
 2180: 
 2181:     use Log::Abstraction;
 2182: 
 2183:     my $logger = Log::Abstraction->new(
 2184:         logger => sub {
 2185:             my $args = shift;
 2186:             my $msg = $args->{ctx}
 2187:                 ? $args->{ctx}->as_string() . ': ' . join('', @{$args->{message}})
 2188:                 : join('', @{$args->{message}});
 2189:             complain({ message => $msg, person => $args->{ctx} });
 2190:         },
 2191:         ctx => $individual,
 2192:     );
 2193: 
 2194:     my $namer = Genealogy::Relationship::Name->new(logger => $logger);
 2195: 
 2196: =back
 2197: 
 2198: If any handler returns without dying (e.g. a warning-only handler in
 2199: L<Log::Abstraction>), C<name()> returns C<undef>.
