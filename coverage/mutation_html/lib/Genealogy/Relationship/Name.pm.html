 1031: Must be a non-negative integer.

 1283: =head2 Error handling
 1284: 
 1285: Errors are dispatched through the following priority chain:
 1286: 
 1287: =over 4
 1288: 
 1289: =item 1. L<Log::Abstraction> object (preferred)
 1290:

 1292: C<ctx> (typically a C<Gedcom::Individual>), then pass it as C<logger> to
 1293: C<new()>.  On error, this module simply calls C<< $logger->error($msg) >>
 1294: and Log::Abstraction handles ctx forwarding, formatting, and dispatch.
 1295: 
 1296:     use Log::Abstraction;
